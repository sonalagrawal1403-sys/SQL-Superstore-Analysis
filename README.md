# SQL-Superstore-Analysis
SQL analysis of the Kaggle Sample Superstore dataset – cleaning, queries, insights, and case study.

Project Overview

The objective of this project is to analyze retail performance across:
Sales
Profitability
Customer behavior
Product performance
Regional trends
Order patterns
Cross-selling opportunities

The project includes:
✔ Data cleaning (CSV → ASCII-safe UTF-8 → MySQL import)
✔ SQL table creation & schema definition
✔ 20+ analytical SQL queries
✔ Insight generation
✔ Business storytelling & recommendations
